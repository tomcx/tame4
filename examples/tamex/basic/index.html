Basic Example Using the TAME Library for Beckhoff ADS-Webservice

TwinCAT v2.10 Build >= 1324 is needed to use this example!

2 checkboxes, values are written to BOOL variables.
The first one ist set to a %MX address. If you click on a button the input value
will be sent to the PLC. 50 ms after the write request is finished a read request
will follow and the value is shown in the right field.
